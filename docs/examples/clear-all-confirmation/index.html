Addon-style recipe

Clear A Saved Search Safely

A support analyst can remove every filter tag without turning an accidental click into lost work.

Scenario

User goal: clear a complete filter set with an understandable confirmation and predictable focus recovery.

Try it

  • Choose Clear all tags, then press Escape.
  • Open it again and choose Cancel.
  • Confirm the action and listen for one result announcement.
  • Add another tag and confirm the trigger becomes available again.

Example

Add filters for the queue, or clear the complete set.

Clear every filter tag? This cannot be undone.

Developer notes

The wrapper calls the existing API and reuses the component's persistent status region.

const count = instance.getTags().length;
instance.clear();
closeConfirmation();
instance.announce(`All ${count} tags cleared.`);

Accessibility notes

  • Every action is a native button and the temporary controls are removed from navigation while hidden.
  • Opening moves focus to Confirm; closing returns focus to the persistent trigger.
  • Escape cancels only while confirmation is open.
  • No extra live region competes with the component status.

Limitations

This inline pattern suits a local reversible-looking action. More complex consequences may need fuller explanatory content, but not automatically a modal.
