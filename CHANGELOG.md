# Changelog

This package uses Changesets for release notes.

## 1.0.0 - 2026-07-17

### Added

- Framework-agnostic tag input enhancement for native `input` and `textarea` controls.
- Typed ESM entry points for the runtime API, documentation metadata, and development inspector.
- Configurable parsing, validation, transforms, serialization, lifecycle hooks, and form reset handling.
- Optional default CSS with theming tokens, visible focus states, forced-colors support, and reduced-motion handling.
- Static examples, GitHub Pages documentation, and accessibility-focused QA artifacts.

### Accessibility

- Keyboard navigation between the editable field and tag removal controls.
- Persistent status messaging for add, remove, paste, validation, and tag-limit feedback.
- Native form semantics, accessible naming, validation relationships, disabled/read-only states, and focus restoration.

### Notes

- This is the initial public release.
- Consumers should complete testing with their target browsers, devices, assistive technologies, form validation flows, and visual themes.
