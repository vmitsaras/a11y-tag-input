# Multilingual Stress Test

This example exercises IME composition, bidirectional text, emoji, accented Latin text, and long unbroken tag values.

Run `npm run build`, then open or serve `examples/multilingual-stress-test/index.html`.

## What to try

- Compose a Japanese or Chinese candidate with an IME; finish composition before pressing Enter.
- Add and remove Arabic and Hebrew tags with keyboard navigation.
- Inspect emoji remove-button names with a screen reader.
- Test the long value at 320 CSS pixels, 200% zoom, and in forced-colors mode.

The samples are test data only. The recipe does not normalize, truncate, or reinterpret values.
